# Next.js Starter prêt pour GitHub et Vercel

## Démarrage local

```bash
npm install
npm run dev
```

## Initialiser Git et pousser sur GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/VOTRE-UTILISATEUR/VOTRE-REPO.git
git push -u origin main
```

## Déployer sur Vercel

1. Connectez votre compte GitHub à Vercel.
2. Dans Vercel, cliquez sur **Add New > Project**.
3. Importez le repository GitHub.
4. Laissez les réglages détectés automatiquement pour Next.js.
5. Cliquez sur **Deploy**.

## Versions utilisées

- Next.js 16.2.1
- React 19.2.0
- TypeScript 5.9.2
